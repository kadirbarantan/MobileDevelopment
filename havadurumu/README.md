# 🌤️ Hava Durumu

Vibe coding yöntemiyle AI destekli olarak geliştirilmiş, sade tasarımlı bir mobil hava durumu uygulaması.

## Teknolojiler

- React Native + TypeScript
- Expo (Expo Go ile test edilebilir)
- Expo Router (navigasyon)
- Open-Meteo API (ücretsiz, API anahtarı gerektirmez)

## Özellikler

- 🌡️ **Anlık hava durumu:** sıcaklık, hissedilen sıcaklık, nem, rüzgar hızı, basınç
- 📅 **5 günlük tahmin:** günlük min/max sıcaklık ve hava durumu açıklaması
- 🔍 **Şehir arama:** dünyadaki herhangi bir şehrin hava durumunu öğrenme
- 📍 **Konum tabanlı:** GPS ile otomatik konum algılama
- 🏙️ **Popüler şehirler:** İstanbul, Ankara, İzmir gibi şehirlere hızlı erişim
- 🌙 **Karanlık/Aydınlık mod:** cihaz temasına uyumlu tasarım

## Kurulum ve Çalıştırma

```bash
# Bağımlılıkları yükle
npm install

# Uygulamayı başlat
npx expo start
```

Açılan QR kodu telefonunuzdaki **Expo Go** uygulaması ile tarayarak uygulamayı çalıştırabilirsiniz.
