# 🧮 Hesap Makinesi

Vibe coding yöntemiyle AI destekli olarak geliştirilmiş, sade tasarımlı bir mobil hesap makinesi uygulaması.

## Teknolojiler

- React Native + TypeScript
- Expo (Expo Go ile test edilebilir)
- Expo Router (navigasyon)

## Özellikler

- ➕ **Temel 4 işlem:** toplama, çıkarma, çarpma, bölme
- 🔬 **Gelişmiş işlemler** (iOS hesap makinesi tarzı toggle panel): `sin`, `cos`, `tan`, `√`, `x²`, `x³`, `x!`, `ln`, `log`, `1/x`, `eˣ`, `10ˣ`, `π`, `e`

## Kurulum ve Çalıştırma

```bash
# Bağımlılıkları yükle
npm install

# Uygulamayı başlat
npx expo start
```

Açılan QR kodu telefonunuzdaki **Expo Go** uygulaması ile tarayarak uygulamayı çalıştırabilirsiniz.

## Geliştirme

Bu proje bilgilerimin üzerine ek olarak, **vibe coding** yaklaşımıyla, AI asistanı kullanılarak geliştirilmiştir.
